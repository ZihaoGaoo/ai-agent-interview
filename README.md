
## 核心技术栈

| 类别 | 技术 |
|------|------|
| 编程语言 | Python, Go, Java |
| 大模型 | Transformer, LLM, LangChain, LlamaIndex |
| AI Agent | RAG, Agent Framework, MCP, A2A |
| 后端 | NestJS, Node.js |
| 前端 | Vue 3, TypeScript |

## 项目结构

```
ai-agent-interview/
├── frontend/                 # Vue 3 前端面试题库
├── backend/                  # NestJS 后端 API
├── docs/                     # 面试题整理
│   ├── 01-python/           # Python 基础
│   ├── 02-llm/              # 大模型基础
│   ├── 03-agent/            # AI Agent
│   ├── 04-rag/              # RAG 技术
│   ├── 05-system/           # 系统设计
│   └── 06-coding/           # 代码题
└── README.md
```

## 快速开始

### 前端启动
```bash
cd frontend
npm install
npm run dev
```

### 后端启动
```bash
cd backend
npm install
npm run start:dev
```

## 重点

1. **Python 编程**: 扎实的基本功
2. **大模型技术**: Transformer, LLM 微调, 推理优化
3. **AI Agent**: RAG, LangChain, LlamaIndex, MCP 协议
4. **系统设计**: 多智能体协作, 分布式架构
5. **算法基础**: 数据结构, 算法, 操作系统
